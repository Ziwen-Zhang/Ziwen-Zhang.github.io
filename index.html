Airdnd

An Airbnb clone.

Airdnd is a full-stack web application designed to replicate core functionalities of Airbnb, including user authentication, dynamic property listings, and reviews.

The backend developed with Express.js, includes RESTful APIs with robust input validation, session-based authentication using JWT, and a normalized SQLite3 database schema implemented with Sequelize. The schema leverages many-to-many relationships through join tables and enforces data integrity with unique and foreign key constraints.

On the frontend, React components with controlled forms enable intuitive user interactions for managing properties, reviews, and accounts.

  • Built with: JavaScript, React, Redux, SQLite3, PostgreSQL
  • Live Link: Airdnd

Ask Geeves

A stack overflow clone

Ask Geeves is a Stack Overflow-inspired web application built to facilitate user interactions through question and answer forums.

The backend, developed with Flask, features a highly relational database schema powered by SQLAlchemy, implementing efficient many-to-many relationships through join tables and ensuring data integrity using custom decorators for existence and authorization checks.

The frontend, crafted with React and TypeScript, offers a dynamic and interactive user experience, leveraging React Router for nested routing and Redux Toolkit with selectors and thunks for scalable state management. Robust RESTful APIs were implemented in Flask, complemented by custom error handlers to streamline validation, authentication, and authorization, ensuring a seamless and secure application workflow.

  • Bulit with: Python, Flask, TypeScript, React, Redux, SQLAlchemy
  • Live Link: Ask Geeves

The Wolf of Wall Street Simulator

A stock trading website.

The Wolf of Wall Street Simulator is a real-time stock trading simulation platform that offers users an immersive experience in the financial market.

The backend, built with Flask, employs SQLAlchemy for managing complex relationships and leverages threading and scheduled background tasks to execute orders and update stock prices and user net worth seamlessly. The platform features dynamic stock price updates powered by WebSocket, ensuring live data streaming for stock prices and notifications.

On the frontend, React and Redux manage the state while Chart.js provides real-time visualizations of stock performance. Together, these technologies enable live updates to user holdings, transaction history, and net worth, creating a comprehensive and interactive trading environment.

  • Built with: Python, Flask, JavaScript, React, Redux, SQLAlchemy, WebSocket
  • Live Link: The Wolf of Wall Street Simulator
